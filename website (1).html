Game

Fatastical End is a game made to acknowleged and talk about mental health. Fantastical end has three different path each dealing with a different person, setting, symphons and illnesss. Each path has a pre selected set of monsters that that path will fight, each of those monsters represent symphons of the mental illiness that that path reprsents.

The paths are a Schizophrenia homeless person, a teen with PTSD, and a middle age man dealing with DID.

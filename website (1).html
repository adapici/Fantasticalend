Game

Fatastical End is a game made to acknowleged and talk about mental health. Fantastical end has three different path each dealing with a different person, setting, symptoms and illnesss. Each path has a pre selected set of monsters that that path will fight, each of those monsters represent symphons of the mental illiness that that path reprsents.

In the game, you can chose to be a homeless person with schisophrenia, a teen with PTSD or a middle-age man with DID. You fight monsters that symbolyze anxiety, paranoia and reality.
